# 组合方法 v0 — 概念 → 水墨 SVG

目标：把一个概念画成一张「像小黑」的极简黑墨 SVG，交给 harness 沙画化。

Task 1 的 `harness.html` 已经建好外壳：把这里产出的 SVG 字符串整段替换 `CONCEPT_SVG`，
用 `Sumi.imageReveal(canvas, img, { n: 7000, seed: 7, alt: caption })` 渲染成粒子场。
本文档只管"画什么、怎么画、交付什么格式"；渲染管线不在本文档范围内。

## 1. 立意（概念 → 视觉隐喻，引导式，不查表）

逐一问自己：
- 这个概念的**核心张力**是什么？（起点 vs 终点 / 卡住 vs 通了 / 分散 vs 汇聚 / 小 vs 大）
- 能用**一个圆头小人的什么动作或处境**表达它？（够不到、推、播种、抱头、伸手、奔跑…）
- **焦点**落在画面哪个位置？靠哪片**留白**把它衬出来？
- 一句话能不能说清这张图在讲什么？说不清就砍到能说清。

这四问按顺序做，不要跳步：先锁张力，再找动作，再定焦点与留白，最后用"一句话测试"验收。
任何一步答不出来，就说明立意还没想清楚——退回上一步，不要带着模糊的立意进入构图阶段。

**"一句话测试"**是唯一的把关标准：把最终 SVG 拿给没看过概念名的人，他们应该能用一句话
说出大意（哪怕说得不精确）。说不出来，就是元素太多、焦点不明确，回到第 3 步重新做减法。

## 2. 原语库 v0（用到才补）

- **圆头小人**：实心圆头 + 躯干（粗笔触/水滴形）+ 四肢（锥形笔触，round cap）。
  姿势预设起步集：站立、伸手向上、坐姿抱头/托腮、奔跑。
- **物件原语（起步集）**：seed/圆点、上扬生长曲线、由疏到密的层叠弧(stacked arcs)、
  箭头、连接线、星芒/火花(spark burst)。
- 需要新原语时：在此登记（名字 + 一句用途），保持"用到才收"。

登记格式（Task 3 画到新原语时，直接在本节追加一行）：

| 原语名 | 一句用途 | 首次用于 |
|---|---|---|
| filled stem wedge（实心生长茎） | 上扬生长茎做成"实心楔形"而非描边——粒子化更稳（细描边会散成噪点） | compound-growth |
| leaf-blade（实心叶片） | 单片实心叶，多片由疏到密堆成树冠，表达"越长越繁茂" | compound-growth |
| connection spoke（实心连接楔） | 由外围指向中心的粗实心楔形，替代细连线，把个体连成网 | silo-to-synergy |
| central hub（中心节点圆盘） | 大实心圆盘作为"汇聚/协同"焦点，各连接楔收束于此 | silo-to-synergy |

原语库自底向上生长：不要在这里预先设计"完整"的形状字典。Task 3 画三张图（复利增长 /
各自为战→协同 / 顿悟）过程中缺什么形状，就在这张表里补一行，再回到画面里用。

## 3. 构图铁律（小黑 DNA）

- 纯黑墨 `#1e2124`，无彩色 / 无渐变。
- **大量留白**、单一焦点，可偏心。
- **毛笔感**：笔触 round cap + 轻微宽度变化；实心形状边缘允许轻微不规则，别几何完美。
- **不带字**（含义靠画，不靠标注）。
- 一张只讲**一个**可读的想法。

补充判断标准（帮助落地上面五条，不是新增规则）：
- 画面元素数量宁少勿多：能用一个圆头小人 + 一到两个物件原语讲清楚，就不要再加第三个物件。
- "留白"不是背景空着，而是主体之外的空间要主动服务焦点——留白面积通常应大于墨迹面积。
- 偏心构图优先于居中：主体放在画面 1/3 附近，比正中心更接近"小黑"的经营感。
- 检查笔触宽度：任何单笔画最窄处也要能撑起一片可辨认的实心色块（见第 4 节"实心面积"约束）。

## 4. SVG 输出契约（喂给 harness 的硬约束）

- `<svg xmlns="http://www.w3.org/2000/svg" width="800" height="800" viewBox="0 0 800 800">`
  —— 显式 `width`/`height` + 近方形 `viewBox`（sumi 采样进 1024² 方形）。
- 只用矢量形状（`path`/`circle`/`rect`/`line`…）；**禁止外链资源、禁止 `<foreignObject>`**
  （会污染 canvas，导致 `getImageData` 抛异常）。
- 透明底即可（sumi 自动垫 `#f4f3ee` 纸底）。
- **别用过细笔画**（粒子化会散成噪点）；主体留足**实心面积**，剪影要 hold 得住。
- 交付 = 一段 SVG 字符串，直接替换 harness 里的 `CONCEPT_SVG`。

落地检查表（每张图定稿前过一遍）：
- [ ] `width`/`height`/`viewBox` 三者都在，且 viewBox 接近正方形（如 800×800）。
- [ ] 全文没有 `<image>`、`<use xlink:href="http://...">`、`<foreignObject>`、外部字体引用。
- [ ] 所有描边（`stroke`）宽度或填充色块的最窄处，肉眼可辨、经得起缩小成粒子场。
- [ ] 颜色只有 `#1e2124`（可用不同透明度表达层次，但不引入第二种色相）。
- [ ] 图内没有 `<text>`；含义完全靠形状和构图传达。
